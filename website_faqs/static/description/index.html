Website FAQ

Answer the FAQs, and help your Website Users

Few Words About The Website FAQ:

By using this application you can answer those questions, that you think these can be the most probably asked questions by your website users or visitors. This will increase the better experience of your website users and visitors.

Work Flow: How The Website FAQ Works
FAQs :
  • List view of all available FAQs for different website
Create a FAQ:
  • Provide all the required fields to create a FAQ
FAQ Created:
  • Click the "Publish" button to make the FAQ visible on the specified website
  • Click the "Unpublish" button to hide the FAQ from the website
FAQ Category:
  • Create new categories
FAQ Tag:
  • Create new tags
Settings:
  • Select the website on which you want show FAQ menu.
  • Then enable the "Show FAQ Menu" check box to make it visible on selected website
Website FAQ Home Page:
  • Select the specific category of FAQs to filter out
  • Click on the card tags to expand and view the questions
  • Use the search box to search the question
FAQs Category List Visible:
FAQs Category List Invisible:
FAQ Viewed by a Category:
FAQ Answer & Feedback:
  • If the user is satisfied with the answer, he can press 'YES' otherwise 'NO'
  • The feedback will help you to improve the FAQs
FAQ Search by Keyword:
  • Enter the tag or keyword in the search box, it will auto suggest the keyword related FAQs
  • Click on the FAQ to view the answer
